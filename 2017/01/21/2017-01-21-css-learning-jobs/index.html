Margin与padidng的区别

Margin是内容的外边框，位于border外面

padding是内容的内边框，位于border里面和内容之间，决定内容与边框的距离。padding是透明的，没有颜色

定于内容的width和height时要考虑padding、border、margin的宽度

CSS Box Model

这个model是css的模型，将html中的元素作为其中的content进行装修

为何要使用 em 而非 px 来定义字的大小

1em大小就是当前页面字体的默认值，相对于16px，1em更加直观。

  • margin-right、
