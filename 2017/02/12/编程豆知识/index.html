编程豆知识

1.dependent::destroy

has_many :posts,dependent::destroy

意思就是group删除的话，对应group里的post也删除

2.model visual

https://github.com/voormedia/rails-erd/blob/master/README.md
可以自动生成上面所示的模型关联图
方法如下：

  1. brew install graphviz
  2. Gemfile中加入 gem "rails-erd",后重新bundle

3.rake erd
atom项目文件下就会生成资料库关系的图示文件 erb.pdf

3.How to user icon in ERB

Refer to font-awesome-helper,fontawesome cheatsheet,fontawesome example

e.g.. <li><%= link_to("login", new_user_session_path) %></li>

To <li><%= link_to(content_tag(:i, '登入', class: 'fa fa-sign-in'), new_user_session_path) %></li>

4.simple_form

simple_form 是rails 内建的helper，可以帮帮把\r\n翻译成

因为系统存数据的方法断行是\r\n，HTML不懂什么是\r\n，HTML只知道断行符号是

5.Helper

Helper是小方法，views用的小方法，Rails内建很多Helper

自定义Helper都会放在app/helpers

6.render “form”是什么意思？

其实这是 <%= render :partial => “form” %> 的“缩写”，意思是我要引用 Partial (局部）页面的方式。因为我们发现有很大量的程式码重复，所以可以用这样的方式收纳。
它的规则是同目录底下的 _xxxx.html.erb

7.irb rails c

console可以调用资料库，irb只能用来测试代码

8.redirect_to :back

回到之前浏览的页面，不发生跳转
