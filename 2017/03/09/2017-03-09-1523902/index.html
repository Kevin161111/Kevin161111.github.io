xdite直播元学习课第2课

objection

关于今天的课程，你记得什么？
完成了什么？

今天听课xdite老师的元学习第2课，写了篇课程提取，发现自己记这种直播课程能力有点差啊，不过大部分知识点get到了

元学习课第2课

Reflective

你要如何形容今天的情绪
今天的高峰是什么？
今天的低点是什么？

工作的事情，项目处理不够顺利，是思考了一下不是我自己的问题而是团队协作的问题。按照全栈营的标准，根本没有做到公开、透明、妥协的前提。项目团队的人加入了别的团队的人，由于利益的关系没办法做到这3个前提，问了下老板是怎么处理这样的事的，给了我一个大大的hit：沟通的目的不是搞定人，而是搞定你想要达到的目的。关于这点我是很认同的，不过项目还是要继续推进的，做完这个项目我想我大概是无法和他们继续协作下去了。

Interpretive

我们今天学到了什么？
今天最重要的领悟是什么？

今天最重要的领悟是被xdite老师的一句话hit到：上完课不提取=花钱买个脑爽。再次体会到提取练习的重要性。这句话和读书是一个道理的，为什么当时看的东西都没有记住，原因就是只是放到了自己的内存，没有放到硬盘里啊！下次学完课程再不提取练习是不是要损失好多！！既然要学东西，那提取练习就是刚需啊！

Desicisional

我们会如何用一句话形容今天的工作？
有哪些工作需要明天继续努力？

今天大部分是在折腾工作项目上的事，回来没有打代码。突然想到自己是不是在给自己找借口：一切困难都是找借口，有刚需的话什么都能办成！！

明天要把魔改的产品继续打磨，主要在产品展示页做好！！
