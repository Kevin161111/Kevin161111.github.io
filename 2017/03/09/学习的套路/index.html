学习的套路-脑科学认知心理学相关

发表于 2017-03-09 | 分类于 套路

这里有视频：TED-肌肉记忆

大脑其实是不善于思考的，多数思考只是调用记忆，也就是习惯回路。

平时大脑都是靠习惯，做过某件事形成记忆回路，让身体去记住某件事。

填鸭教育有时候是必须的，无重复练习不可能真正精通一项技能。没有大量的重复练习，不可能精通任何的脑力活动。

因为没有形成习惯回路嘛。

复杂状况内形成直觉。所谓直觉，就是深植于肌肉里的长期记忆。

多数的思考并不是真正的思考，而是调用记忆形成习惯。

人类无法用未知解释未知，只能用已知来解释未知。如何获取已结-习惯回路的形成。这个内容和《习惯的力量》里面讲的相同。

肌肉记忆是什么：

  • 提取练习
  • 迁移练习

提取练习是什么：

  • 把书合起来，笔记合起来
  • 回想重点并写下来
  • 对应上一份笔记
  • 重复讲给别人听，教会你身边的人

经过上面的步骤才是有效的提取练习，因为在不知不觉中形成了你自己的认知套路

教会别人=创造自己的一套学问模板

上完课不提取=花钱买个脑袋爽

迁移练习是什么？

先说一个新手运的东西

  1. 高效得到，从某处得到一个方法或者套路
  2. 以为自己天资聪颖，简单，我还很厉害的，什么都搞得定
  3. 用不出来，写不出来，最终眼高手低

以上说的就是新手运，新手在接触新的知识的时候大部分的历程是这样的。高效得到，你学的是大量老手总结的知识点，这些知识点已经包含了大量的学习过程中的坑。所以新手才没那么多的坑可踩，反而会觉得简单。其实那些坑才是学的到的关键啊，经历了坑才会有自己的思考过程，才会形成自己的套路啊。由此可见，学习过程中的错误记录多么重要！！

并不是说新手运不好，大部分学新的东西都要经历上面的步骤，那么如何形成迁移：

大量的重复，用上5遍10遍，遍历所有的坑，对各种状况都经历过，写出自己的提取，教会别人-转移新手运给他们。

学知识（技能）的小套路

  1. 初学时，取得技能的心态，我就是要学会XXX，可以用就行了，不必追求完美，那是以后的事

  2. 快速取得成就感，可以不理解，可以抄，可以不扎实，可以粗糙。可以是野路子，不需要别人认可的野路子。只要学得会想要的技能都可以。做出来，建立small win。

  3. 用熟，大量的重复和迁移，形成肌肉记忆，别忘了前面说的肌肉记忆是什么。重复：大量练习，教会别人。迁移：只有能迁移的才是你自己的。不迁移只是图个脑爽而已，再次掉进新手运而已。

    ​

小套路升级到大套路

  1. 找到大牛的套路，找到新手运
  2. 练熟，形成肌肉记忆，能迁移。
  3. 做提取，教会别人
  4. 练习小的项目
  5. 迁移到中套路
  6. 中套路的肌肉记忆，迁移
  7. 迁移到大套路，至此才算入门了，找到自己的兴趣主题。
